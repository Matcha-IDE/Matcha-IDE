Goal: Annotate other data transmissions not covered by the API call detection


Double-click "Annotate other data transmission variables" to expand the list.


Double-click on "Search for possible data transmission" to show keyword search results.


If your app sends user data out of the device or shares it with other apps on device, annotate the data transmission request.


Quickfix for adding the annotation: click on the target variable and press [Alt]+[Enter] (for Windows/Linux) or [Option]+[Enter] (for Mac) to show the quickfix list. Click "Annotate data transmission behavior" to apply the quickfix
