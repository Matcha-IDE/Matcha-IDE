Goal: Annotate user data transmitted by API calls detected by Matcha


Double-click "Add required annotations" to see all the detected API calls that are not correctly annotated. Then click on each item to see the API call and use quickfix to add the annotation.


How to use quickfix: click on the error in code and press [Alt]+[Enter] (for Windows/Linux) or [Option]+[Enter] (for Mac) to show the quickfix list. Click "Annotate data transmission behavior" to apply the quickfix
