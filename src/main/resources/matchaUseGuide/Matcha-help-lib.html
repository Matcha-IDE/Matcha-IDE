Goal: Verify data collected or shared by 3rd-party libraries


Double-click "Verify custom 3rd-party SDK data usage" to see all the unverified libraries.


Click each item to see the detailed data usage for each library in the XML config file auto-generated by Matcha.


Read the text enclosed by the <data> tag to check if certain data usage is applicable to your app. Novice users can ignore the attributes of the tag.


Remove or keep each <data> tag depending on your usage. Then set the "verified" attribute to true.


You can also click "Review verified custom 3rd-party SDK data usage" and "Review default library data usage" to see data automatically collected or shared by default and can't be turned off.
